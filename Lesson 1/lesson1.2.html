Квадрат числа: a2

Радус сфери: r = 3√(V/(4π/3))

Формула квадратного кореня: n√(an) = a

Формула квадратного рівняння: ax2 + bx + c = 0

Формула знаходження кореня квадратного рівняння: x=(-b ± √(b2) - 4ac)) / 2a

Площа кола: S = πr2

Теорема Піфагора: c = √(a2 + b2)
